# Network Security: Phishing Data Prediction

An end-to-end Machine Learning project designed to classify and predict phishing attacks in network data. This application features a robust machine learning pipeline and a FastAPI-based backend for seamless model training and real-time bulk predictions.

## 🚀 Features
* **Automated Data Pipeline**: Handles modular data ingestion from MongoDB, validation, transformation, and model evaluation.
* **Model Training Endpoint**: Trigger a full model re-training pipeline through a simple API call.
* **Batch Prediction API**: Upload CSV files to receive immediate network security threat predictions.
* **Interactive Web Interface**: View your prediction results via a cleanly formatted, responsive HTML dashboard using Jinja2.
* **API Documentation**: Built-in Swagger UI provided by FastAPI for easy API testing.

## 🛠️ Tech Stack
* **Backend Framework**: FastAPI, Uvicorn
* **Machine Learning**: Scikit-Learn, Pandas
* **Database**: MongoDB (via `pymongo`)
* **Frontend Rendering**: Jinja2 Templates, HTML/CSS (Bootstrap 5)
* **Environment Configuration**: `python-dotenv`
